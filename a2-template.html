Dataset

The Data set used was a cleaned version of assignment 1's data set
The data set gives insight into DFW rates of all students from F17 - S22. The cleaned version is a much simpler version to use for application like tableau. This data was produced by Sacramento State University and was cleaned by Muhammad Ahmed. We are interested in this data because it gives us insight into how covid has affected the DFW rates of students in the ECS department.

The data is available online at: https://csus.instructure.com/courses/107503/pages/examples-of-cleaning-assignment-1-dataset?module_item_id=5388593

Summary

Some of the lessons learned are based on choosing data wisely and using it efficiently to convey an answer to the questions at hand. We all decided to use bar graphs as we all agreed it is the easiest to read when trying to convey a message using the data. The only concerns we seemed to have at times was being able to answer the covid questions rates with only the range given as data is best interpreted with more data and the whole range included. Our best answer to this was sticking with the individual graph ranges to be interpreted and analyzed separately.
