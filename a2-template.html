Dataset

The Data set used was a cleaned version of assignment 1's data set
The data set gives insight into DFW rates of all students from F17 - S22. The cleaned version is a much simpler version to use for application like tableau. We are interested in this data because it gives us insight into how covid has affected the DFW rates of students in the ECS department.
